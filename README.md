# This code is to replicate the analysis in the paper "Seeing Through Green: Text-Based Classification and the Firm's Returns from Green Patents"

The data supporting the findings of this study are from PATSTAT Global (EPO) and ORBIS IP database (Bureau Van Dijk). Access to these data requires a valid license and they are not publicly available.

If you use the code or data, please cite the following paper: "Seeing Through Green: Text-Based Classification and the Firm's Returns from Green Patents".

The NLP analysis was conducted in python while the econometric application in Stata, hence the code is separated in two different folders.
